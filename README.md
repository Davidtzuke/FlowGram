# FlowGram

Buffer management for terminal users.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/Davidtzuke/FlowGram/main/flowgram.zsh -o ~/.flowgram.zsh && source ~/.flowgram.zsh --install
```

## Keys

| Key | Action |
|-----|--------|
| Ctrl+A | Select all |
| Ctrl+K | Nuke (clear with undo) |
| Ctrl+Z | Restore |

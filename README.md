# FlowGram

Buffer management for terminal.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/Davidtzuke/FlowGram/main/flowgram.zsh | zsh -s -- install
```

Then restart terminal or `source ~/.zprofile`

## Keys

| Key | Action |
|-----|--------|
| Ctrl+A | Select all |
| Ctrl+K | Nuke (clear + save) |
| Ctrl+Z | Restore |
